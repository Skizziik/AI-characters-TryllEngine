Background collage tiles for the landing / onboarding hero.

Drop files here named: c01.png, c02.png, ... c15.png
Recommended: vertical 3:4, ~768 x 1024, compressed (these are darkened + blurred
behind the hero, so light weight is fine — avoid multi-MB files).

The collage component auto-tiles whatever cNN.png files exist into slow-drifting
columns with a bordeaux/black scrim on top. Missing files are simply skipped.
